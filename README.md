# 🏡 SriKrishna Real Estate

A modern, premium, and fully responsive real estate website for **SriKrishna Real Estate**, located in **Bilichy Village, Karamadai, Coimbatore, Tamil Nadu, India**.

The website is designed to showcase premium residential and commercial properties with a clean, elegant, and user-friendly interface. It focuses on building trust, improving customer engagement, and generating quality leads.

---

## ✨ Features

- Modern and premium UI/UX
- Fully responsive design
- Property listings with images
- Property details page
- Search and filter functionality
- Contact form for inquiries
- Admin login system
- Secure authentication using JWT
- Mobile-friendly layout
- Fast and optimized performance

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/sanjays2007/SriKrishna-realestate.git
```

Go to the project directory

```bash
cd SriKrishna-realestate
```

Install dependencies

```bash
npm install
```

Create the environment file

```bash
cp .env.example .env
```

Start the server

```bash
npm start
```

---

## 🔐 Environment Variables

Create a `.env` file and configure the following:

```env
PORT=3000
JWT_SECRET=your_secret_key
ADMIN_USERNAME=admin
ADMIN_PASSWORD=your_password
```
